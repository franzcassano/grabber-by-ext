# grabber-by-ext
Scraping Domain List By Domain Extension

## Usage:
<li>python 3.11.1</li>
<li>using cmd/terminal</li>

## Common Modules:
<li>requests</li>
<li>colorama</li>
<li>pystyle</li>
<li>bs4</li>

## Features:
<li>Auto Remove Duplicate.</li>
<li>Auto Scrape Until The Latest Page</li>
<li>More Than 100+ Domain Extension</li>
